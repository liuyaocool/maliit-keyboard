# Install

```bash
git clone https://github.com/liuyaocool/maliit-keyboard.git --depth=1
cd maliit-keyboard
mkdir build
cd build
cmake ..
make
sudo make install

# replace
sudo cp /usr/local/bin/maliit-keyboard /usr/xxx/maliit-keyboard
```
